🎮 Jogo da Força

Projeto desenvolvido com o objetivo de praticar e evoluir habilidades em programação Python através da implementação do jogo da força clássico.

📚 Sobre o projeto

Este jogo foi inicialmente criado com o auxílio de Inteligência Artificial, e posteriormente analisado, revisado e aprimorado manualmente, com foco em aprendizado e melhoria contínua.

A proposta não foi apenas utilizar o código gerado, mas entender sua lógica, corrigir pontos necessários e adaptá-lo conforme meu próprio estilo de desenvolvimento.

🎯 Objetivos Reforçar conceitos de lógica de programação Praticar desenvolvimento em Python Aprender a interpretar e melhorar códigos gerados por IA Desenvolver autonomia na resolução de problemas 🚀 Funcionalidades Geração de palavra secreta Entrada de letras pelo usuário Controle de acertos e erros Condição de vitória e derrota 🖥️ Tecnologias utilizadas Python
